# Event Storming

## Concepts

### Command

> Decision or action

Colour: **Blue**

### Aggregate

> Local logic.

Colour: **Yellow**

### Event

> Domain Event

Colour: **Orange**

### Policy

> Reactive Logic (Whenever), Listener, Process Manager, Saga

Colour: **Lilac**

### Read Model

> Key Information

Colour: **Green**

## Basic Flow

Command -> Aggregate -> Event -> Read Model or Policy.
